# FEATURE GATING LOGIC SUMMARY

## Pages Available by Tier

### Free (all roles)

Dashboard, My Profile/Opportunities (limited), Marketplace (browse), Browse Brands (limited), Connect Accounts (2 platforms for talent, full for brand), Deal Pipeline (1 deal), Contract Templates (view), Basic Analytics, Master Calendar (view), Culture Calendar (view), Referrals, Notifications, Settings, Billing, Subscriptions

### Paid Tier 1 (Rising $99 / Growth $499 / Agency Starter $2,499)

Everything in Free PLUS: Match Engine (basic), Outreach (limited), Full marketplace access, More active deals, Contract use, Basic talent/campaign analytics, Calendar reminders

### Paid Tier 2 (Pro $249 / Scale $1,299 / Agency Pro $4,999)

Everything in Tier 1 PLUS: Full Match Engine, Unlimited outreach + sequences, Contact Finder, Warm Intro, Demographic Targeting, All deal features, Market Intelligence, Spend Prediction, ROI Simulator, Pitch Deck Builder, Data Room, AI Command Center, AI Agents Hub, Custom Reports, Teams

### Paid Tier 3 (Elite $499 / Enterprise $2,500+ / Agency Enterprise $9,999+)

Everything in Tier 2 PLUS: Auto-matching, Unlimited everything, Priority placement, Dedicated support, White-label, API access, SSO, SLA

# WHY EACH FEATURE IS ON ITS TIER

| Feature                 | Tier                       | Reasoning                                                 |
| ----------------------- | -------------------------- | --------------------------------------------------------- |
| **Dashboard**           | Free                       | Core experience — everyone needs it to stay engaged       |
| **My Profile**          | Free                       | Must exist to be discoverable — no profile = no value     |
| **Connect Accounts**    | Free (limited)             | Let them taste verification, limit creates upgrade desire |
| **Marketplace**         | Free (browse)              | Seeing opportunities they can't act on drives upgrades    |
| **Match Engine**        | Paid                       | #1 upgrade driver — the core AI value proposition         |
| **Contact Finder**      | Paid (Tier 2)              | High-value feature — finding the right person to email    |
| **Outreach**            | Paid (Tier 1)              | Can receive free, but sending requires payment            |
| **Sequences**           | Paid (Tier 2)              | Advanced automation — power users only                    |
| **AI Features**         | Paid (Tier 2)              | Expensive to run (API costs), high value                  |
| **Deal Pipeline**       | Free (1 deal)              | Let them complete one deal to experience full value       |
| **Analytics**           | Free (basic) → Paid (full) | Basic stats free, deep analytics paid                     |
| **Referrals**           | Free                       | Viral growth mechanism — NEVER gate this                  |
| **Notifications**       | Free                       | Core UX — never gate communications                       |
| **Settings**            | Free                       | Core UX — never gate account management                   |
| **Warm Intros**         | Paid (Tier 2)              | Network access is premium value                           |
| **Market Intelligence** | Paid (Tier 2)              | Proprietary data = premium product                        |
| **Pitch Deck Builder**  | Paid (Tier 2)              | AI costs + high value output                              |
| **Data Room**           | Paid (Tier 2)              | Sensitive intelligence data                               |
| **Teams**               | Paid (Tier 2+)             | Multi-user = enterprise feature                           |
| **Integrations**        | Paid (Tier 3)              | Enterprise connectivity                                   |
| **White-label**         | Paid (Tier 3)              | Premium customization                                     |
