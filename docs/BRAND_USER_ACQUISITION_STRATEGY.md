# PART 1: MONTH-BY-MONTH ACQUISITION TARGETS

## Rationale for Ramp Curve

Brands have 30–90 day sales cycles for paid tiers. Free-tier acquisition can happen in days. The growth curve is back-weighted because:

1. SEO content takes 3–4 months to rank
2. Outbound sequences need 6–8 touchpoints over 3–4 weeks
3. Enterprise deals require 60–90 day evaluation cycles
4. Referral loops don't activate until Month 4–5 when early users have success stories
