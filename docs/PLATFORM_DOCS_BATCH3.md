# DELIVERABLE 8: DEPENDENCY & BLAST RADIUS MAP

## Overview

This map documents every external and internal system dependency in DealStage with blast radius analysis. Criticality levels indicate the severity of a complete failure of that dependency:

| Criticality | Meaning                                                           | Response Time Target |
| ----------- | ----------------------------------------------------------------- | -------------------- |
| 1           | Platform-down: total outage, all users affected                   | < 15 minutes         |
| 2           | Major degradation: core features broken, workarounds limited      | < 1 hour             |
| 3           | Partial degradation: some features broken, workarounds exist      | < 4 hours            |
| 4           | Minor impact: non-critical feature affected, users may not notice | < 24 hours           |

### Revenue Impact Calculation

Revenue impact is estimated based on projected MRR of $50,000 at Year 1 scale (5,000 users), calculated as:

- **$/hour** = MRR / 730 hours/month = ~$68.50/hr base
- Adjusted by percentage of users affected and percentage of revenue-generating flows blocked
